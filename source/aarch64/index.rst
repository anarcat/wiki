.. _aarch64:

AARCH64 (arm64)
===============

Ampere Computing has loaned to the OSUOSL a cluster of aarch64 machines that have KVM capability enabled on them. These
are development machines and are not available for any use other than to support FOSS.

These machines are intended to run Openstack in a cluster fashion powered by KVM.
